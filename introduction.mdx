## What is Los Simpson?

Los Simpson is a modern React web application that brings The Simpsons universe to your fingertips. Browse through hundreds of characters, explore episodes from all seasons, and discover detailed information about your favorite Springfield residents.

## Key Features

<AccordionGroup>
  <Accordion title="Comprehensive Character Database" icon="address-book">
    Access information on hundreds of Simpsons characters including their age, gender, status, and occupation. Each character includes a high-quality portrait from The Simpsons API.
  </Accordion>
  
  <Accordion title="Episode Information" icon="film">
    Browse through all episodes with details including air dates, season numbers, episode numbers, and synopses. Navigate through 39 pages of episode content.
  </Accordion>
  
  <Accordion title="Responsive Design" icon="mobile">
    Enjoy a seamless experience on any device with a fully responsive layout that adapts to desktop, tablet, and mobile screens.
  </Accordion>
  
  <Accordion title="Smart Pagination" icon="arrows-left-right">
    Efficiently navigate through large datasets with built-in pagination controls and loading states for smooth transitions.
  </Accordion>
</AccordionGroup>

## Technology Stack

Los Simpson is built with modern web technologies:

- **React 19** - Latest version with improved performance and features
- **TypeScript** - Type-safe code for better maintainability
- **Vite** - Lightning-fast build tool and dev server
- **React Router** - Client-side routing for seamless navigation
- **CSS Modules** - Scoped styling to prevent conflicts
- **The Simpsons API** - Real-time data integration
