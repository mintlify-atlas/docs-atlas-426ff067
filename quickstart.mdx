Get Los Simpson up and running on your local machine in just a few minutes. This guide will take you from cloning the repository to viewing the character gallery in your browser.

## Prerequisites

Make sure you have [Node.js](https://nodejs.org/) installed on your system. Los Simpson works with Node.js 18 or higher.

## Quick Start

<Steps>

<Step title="Clone the repository">

Clone the Los Simpson repository to your local machine:

```bash
git clone https://github.com/your-username/los-simpson.git
cd los-simpson
```

<Note>Replace `your-username` with the actual GitHub username or organization.</Note>

</Step>

<Step title="Install dependencies">

Install the required dependencies using your preferred package manager:

<CodeGroup>

```bash npm
npm install
```

```bash pnpm
pnpm install
```

```bash yarn
yarn install
```

</CodeGroup>

This will install React, TypeScript, Vite, React Router, and all development dependencies.

</Step>

<Step title="Start the development server">

Launch the Vite development server:

<CodeGroup>

```bash npm
npm run dev
```

```bash pnpm
pnpm dev
```

```bash yarn
yarn dev
```

</CodeGroup>

Vite will start the development server and display the local URL in your terminal.

</Step>

<Step title="Open the application">

Open your browser and navigate to:

```
http://localhost:5173
```

You should see the Los Simpson character gallery with an interactive grid of characters from the iconic TV series. You can browse through 60 pages of characters, view their details including age and status (alive or deceased), and navigate using the pagination controls.

<Tip>The port may differ if 5173 is already in use. Check your terminal output for the actual URL.</Tip>

</Step>

</Steps>

## What's Next?

Now that you have Los Simpson running, you can:

- Browse through the character gallery
- Click on individual characters to see their details
- Use the pagination controls to explore all characters
- Explore the codebase to understand the React + TypeScript architecture

## Available Scripts

Here are the main scripts you can run:

- `dev` - Start the development server with hot reload
- `build` - Create a production build with TypeScript compilation
- `preview` - Preview the production build locally
- `lint` - Run ESLint to check code quality

<Note>Los Simpson uses [The Simpsons API](https://thesimpsonsapi.com/) to fetch character data. No API key or configuration is required.</Note>
